FAQ

How long does it take?

Usually between 3 to 7 days, depending on the complexity and scope of the project. I’ll let you know the estimated timeline once we discuss the details.

Will the website be mobile-friendly?

Absolutely! Every website I build is fully responsive, meaning it adapts perfectly to phones, tablets, and desktop screens.

What do you need from me to start the project?

To get started, I usually need a design file (like Figma, PSD, or PDF) or at least a clear idea of what you're looking for. You’re also welcome to send any content you already have — such as text, images, product info, or brand assets. And if you don’t have a design yet, no worries — I can help you create one from scratch.

Can I edit the website myself, and will I know how to use it?

Yes – if the site is built using WordPress or Shopify, you’ll have access to an easy-to-use admin panel where you can update content, images, and products. I’ll also guide you through the basics or provide a short tutorial, so you feel confident managing the site yourself.

Can I request changes after delivery?

Yes, I’m always open to feedback and adjustments. I’ll make sure you're satisfied with the final result before wrapping up the project.
